Getting Started

Welcome to RideBook, the open-source bicycle rental system. This documentation will guide you through the setup, configuration, and usage of RideBook.

System Requirements

  • PHP 7.4 or higher
  • MySQL 5.7 or higher
  • Web server (Apache, Nginx, etc.)
  • Composer (PHP package manager)
  • Node.js and npm (for frontend development)

Installation

  1. Clone the repository: git clone https://github.com/Mohammed12des/Bicycle-Rental-System.git
  2. Navigate to the project directory: cd Bicycle-Rental-System
  3. Install PHP dependencies: composer install
  4. Install JavaScript dependencies: npm install
  5. Copy the .env.example file to .env and configure your database settings
  6. Generate an application key: php artisan key:generate
  7. Run database migrations: php artisan migrate
  8. Seed the database with sample data: php artisan db:seed
  9. Start the development server: php artisan serve

Configuration

RideBook can be configured through the .env file and config files in the config directory. Key configuration options include:

  • Database settings
  • Mail server settings
  • Payment gateway integration
  • Google Maps API key (for location services)

Refer to the comments in the configuration files for detailed explanations of each option.

User Guide

Admin Dashboard

The admin dashboard allows you to manage bicycles, users, bookings, and view reports. Key features include:

  • Adding and managing bicycle inventory
  • User management
  • Booking overview and management
  • Generating reports

Customer Interface

The customer interface allows users to:

  • Browse available bicycles
  • Make reservations
  • View and manage their bookings
  • Update their profile

API Reference

RideBook provides a RESTful API for integration with other systems. Key endpoints include:

  • GET /api/bicycles: List all bicycles
  • POST /api/bookings: Create a new booking
  • GET /api/users/{id}: Get user details

For a complete API reference, please refer to our API documentation.

Troubleshooting

If you encounter issues while using RideBook, try the following steps:

  1. Check the error logs in storage/logs/laravel.log
  2. Ensure all dependencies are installed and up to date
  3. Verify your configuration settings in the .env file
  4. Clear application cache: php artisan cache:clear

If you still need assistance, please open an issue on our GitHub repository or reach out to our community support channels.
